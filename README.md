# 🚀 太空射击游戏 (Space Shooter)

一个使用纯HTML5 Canvas和JavaScript构建的网页小游戏，任何人都可以访问和游玩。

## 🎮 游戏特点

- **经典太空射击玩法**：控制飞船，消灭下落的陨石
- **响应式设计**：支持桌面和移动设备
- **完整游戏系统**：分数、生命值、等级、连击奖励
- **本地存储**：保存最高分记录
- **音效系统**：使用Web Audio API创建游戏音效
- **无需安装**：直接在浏览器中游玩

## 🎯 游戏控制

### 桌面端
- **← → 键**：左右移动飞船
- **空格键**：发射子弹
- **P 键**：暂停/继续游戏
- **Enter 键**：开始/重新开始游戏

### 移动端
- 使用屏幕上的按钮控制：
  - **← 左移** / **右移 →**：移动飞船
  - **🔥 射击**：发射子弹

## 📊 游戏规则

1. **目标**：保护太空，消灭尽可能多的陨石
2. **得分**：
   - 每消灭一个陨石：+10分
   - 连击奖励：连续消灭有额外加分（最多5连击）
3. **生命值**：
   - 初始3条命
   - 被陨石击中：-1条命
   - 生命值为0时游戏结束
4. **等级系统**：
   - 每得100分升一级
   - 等级越高，陨石下落速度越快

## 🛠️ 技术栈

- **HTML5**：游戏结构和界面
- **CSS3**：响应式设计和动画效果
- **JavaScript (ES6)**：游戏逻辑和Canvas渲染
- **Canvas API**：2D图形渲染
- **Web Audio API**：游戏音效
- **LocalStorage**：保存最高分

## 🌐 部署到GitHub Pages

### 方法1：通过GitHub网页界面
1. 创建新的GitHub仓库
2. 上传所有文件到仓库
3. 进入仓库设置 → Pages
4. 选择分支（通常是main）和根目录
5. 保存后等待部署完成（通常需要1-2分钟）

### 方法2：通过Git命令行
```bash
# 初始化Git仓库
git init
git add .
git commit -m "Initial commit: Space Shooter game"

# 连接到GitHub仓库
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

部署完成后，游戏将在 `https://你的用户名.github.io/仓库名/` 可访问。

## 🔧 自定义和扩展

### 修改游戏难度
在 `game.js` 中调整以下参数：
```javascript
this.gameSpeed = 3;          // 基础游戏速度
this.asteroidDelay = 1000;   // 陨石生成间隔（毫秒）
this.lives = 3;              // 初始生命值
```

### 添加新功能
- **新敌人类型**：在 `createAsteroid()` 方法中添加新的陨石类型
- **道具系统**：创建道具类并添加到碰撞检测
- **背景音乐**：使用 `<audio>` 元素或Web Audio API添加背景音乐
- **粒子效果**：为爆炸添加粒子系统

## 📱 移动端优化

游戏已包含移动端控制按钮，但可以进一步优化：
